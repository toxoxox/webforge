# WebForge Product Overview

WebForge is a browser-based student website builder that enables students to learn web development by creating, editing, and exporting websites directly in the browser.

## Core Purpose

Educational platform for teaching HTML, CSS, and JavaScript through hands-on website creation with immediate visual feedback.

## Key Features

- **Monaco Editor Integration**: Professional VS Code editor with IntelliSense, syntax highlighting, and code completion
- **Live Preview**: Real-time sandboxed iframe preview of websites as students code
- **Project Management**: Create, save, and manage multiple website projects with auto-save to localStorage
- **Templates**: Pre-built portfolio and blog templates for learning and starting points
- **Tutorials**: Integrated step-by-step guides for HTML, CSS, and JavaScript fundamentals
- **Export**: Download projects as ZIP files ready for hosting
- **No Installation**: Runs entirely in the browser with no build tools or server required

## Target Users

Students learning web development, particularly in educational settings where ease of setup and immediate feedback are critical.

## Design Philosophy

- Pure vanilla JavaScript - no frameworks or build tools
- Zero installation - works immediately in any modern browser
- Educational focus - built-in learning resources and safe experimentation
- Responsive design - works on mobile, tablet, and desktop devices
