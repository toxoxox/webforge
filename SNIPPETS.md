# Code Snippets Library

The Code Snippets Library provides students with a searchable collection of common web development patterns that can be instantly inserted into their projects.

## Features

- **25+ Pre-built Snippets**: Common HTML, CSS, and JavaScript patterns
- **Searchable**: Find snippets by name, description, or category
- **Language Filters**: Filter by HTML, CSS, or JavaScript
- **Click-to-Insert**: Insert snippets at cursor position with one click
- **Organized by Category**: Snippets grouped by purpose (Components, Layout, Forms, etc.)

## How to Use

1. **Open Snippets Panel**: Click the "Browse Snippets" button in the sidebar
2. **Search or Filter**: Use the search box or language filter buttons
3. **Preview Code**: View snippet code before inserting
4. **Insert**: Click "Insert at Cursor" to add the snippet to your file
