# WebForge Component Workshops

## Available Component Workshops

These component workshops are designed to help students build real-world website projects. Each workshop includes step-by-step instructions, code examples, and learning objectives.

### Navigation Components

#### 1. Navigation Bar with Dropdown
- **Difficulty**: Beginner
- **Duration**: 40 minutes
- **Perfect for**: All websites
- **Features**: Logo, menu links, dropdown menus
- **Technologies**: HTML, CSS (Flexbox)

### Layout Components

#### 2. Hero Section
- **Difficulty**: Beginner
- **Duration**: 25 minutes
- **Perfect for**: Landing pages, home pages
- **Features**: Full-screen hero, gradient background, CTA button
- **Technologies**: HTML, CSS (Flexbox)

#### 3. Image Gallery Grid
- **Difficulty**: Beginner
- **Duration**: 35 minutes

#### 4. Timeline Component
- **Difficulty**: Intermediate
- **Duration**: 40 minutes
- **Perfect for**:
  - Gaming history websites

  - Indigenous history websites
  - City history pages
- **Features**: Vertical timeline, event markers, dates
- **Technologies**: HTML, CSS (Flexbox, pseudo-elements)

## How to Use Component Workshops

1. **Open WebForge**: Navigate to `components.html`
2. **Browse Components**: Use filters to find components by type or difficulty
3. **Click "Build This"**: Start the workshop for any component
4. **Follow Steps**: Complete each step in the workshop panel
5. **See Live Preview**: Watch your component come to life in real-time
6. **Copy Code**: Use the finished component in your project

## Next Steps

Students can mix and match these components to build complete websites for their projects. Each component is designed to be:
- **Beginner-friendly**: Clear explanations and step-by-step guidance
- **Responsive**: Works on mobile, tablet, and desktop
- **Customizable**: Easy to modify colors, text, and images
- **Professional**: Production-ready code following best practices
